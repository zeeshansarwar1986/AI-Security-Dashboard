# 👁️ SENTINEYE: Enterprise AI Security Dashboard

**SENTINEYE** is a state-of-the-art security monitoring system leveraging high-performance face recognition and biometric intelligence. Designed for enterprise-grade surveillance, it provides real-time detection, identity tracking, and security analytics through a premium glassmorphic interface.

---

## 🇵🇰 اردو تفصیل (Urdu Description)

**سینٹنی آئی (SENTINEYE)** ایک جدید ترین سیکیورٹی مانیٹرنگ سسٹم ہے جو آرٹیفیشل انٹیلیجنس (AI) اور چہرہ پہچاننے والی ٹیکنالوجی (Face Recognition) پر مبنی ہے۔ یہ سسٹم ریئل ٹائم میں لوگوں کی شناخت کرنے، مشکوک سرگرمیوں پر نظر رکھنے اور سیکیورٹی ڈیٹا کا تجزیہ کرنے کے لیے ڈیزائن کیا گیا ہے۔

* **ریئل ٹائم مانیٹرنگ:** لائیو کیمرہ فیڈ پر فوری شناخت۔

* **سیکیورٹی انٹیلیجنس:** سیکیورٹی گراف اور ڈیٹا اینالیٹکس۔

## 🇬🇧 English Description

### Core Features

* **Live Identity Monitor:** Real-time webcam/CCTV feed with neural network processing for instant face detection.
* **Biometric Database:** Manage personnel, VIPs, and restricted individuals with high-accuracy biometric profiles.
* **Security Analytics:** Advanced data visualization of security events, detection frequency, and node status.
* **Cyberpunk Glassmorphism UI:** A high-contrast, premium interface designed for high-stakes security environments.

### Tech Stack

* **Frontend:** React, TailwindCSS, Chart.js, Lucide Icons.
* **Backend:** FastAPI (Python), SQLModel (SQLite), OpenCV, Face Recognition Library.
* **Design:** Premium Glassmorphism with High-Contrast Design Tokens.

## 🚀 Setup Instructions

### Backend

1. Navigate to `backend/`
2. Create virtual environment: `python -m venv venv`
3. Activate venv: `.\venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Run server: `python main.py`

### Frontend

1. Navigate to `frontend/`
2. Install dependencies: `npm install`
3. Start dev server: `npm run dev`
